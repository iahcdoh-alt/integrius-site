# Documentação

Integração via Webhook (exemplo cURL):

```bash
curl -X POST $WHATSAPP_WEBHOOK_URL \
-H "Content-Type: application/json" \
-d '{"source":"site","event":"lead","payload":{"name":"Exemplo","phone":"+55...","email":"a@b.com","message":"Quero uma demo","product":"10equinze"}}'
```

### Recursos
- **WhatsApp/Twilio**: encaminhamento de leads para seu fluxo n8n.
- **API REST**: rota `/api/lead` para recebimento de leads.
